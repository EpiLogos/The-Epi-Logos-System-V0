# Font Files for Epi-Logos System

## Ranade Font Files

Place the Ranade font files in this directory with the following names:

- `Ranade-Regular.woff2` (400 weight)
- `Ranade-Medium.woff2` (500 weight)  
- `Ranade-SemiBold.woff2` (600 weight)
- `Ranade-Bold.woff2` (700 weight)

## Download Instructions

1. Go to [Google Fonts - Ranade](https://fonts.google.com/specimen/Ranade)
2. Select the weights: Regular (400), Medium (500), SemiBold (600), Bold (700)
3. Download the font family
4. Convert TTF files to WOFF2 format (for better web performance)
5. Place the WOFF2 files in this directory

## Font Usage

- **Ranade**: Used for headings (h1, h2, h3, h4, h5, h6)
- **Work Sans**: Used for body text (automatically loaded from Google Fonts)

## Tailwind Classes

- `font-heading` - Uses Ranade font family
- `font-sans` - Uses Work Sans font family

## CSS Variables

- `--font-ranade` - Ranade font family variable
- `--font-work-sans` - Work Sans font family variable
