## ▲ Quick Start

### Prerequisites
- Node.js 18+ and Python 3.11+
- Neo4j Aura, MongoDB Atlas, Redis Cloud accounts
- Google OAuth credentials
- OpenAI/Anthropic/Google AI API keys

### Development Setup
```bash
# Clone and install dependencies
git clone <repository-url>
cd "The Epi-Logos System V0"
npm install

# Configure environment
cp .env.example .env
# Edit .env with your database credentials

# Start all services (trilaminar architecture)
npm run dev:all

# Or start individually
npm run dev:frontend  # Port 3000 - Experience Vessel
npm run dev:agentic   # Port 8001 - Nervous System
npm run dev:backend   # Port 8000 - Deep Engine Room
```

### Environment Configuration
Create a single `.env` file implementing coordinate-aware configuration:
```env
# Database Constellation
NEO4J_URI=neo4j+s://your-instance.databases.neo4j.io
NEO4J_USERNAME=neo4j
NEO4J_PASSWORD=your-password

MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/
REDIS_URL=redis://username:password@host:port

# Authentication (Nara subsystem integration)
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret
JWT_SECRET=your-jwt-secret

# AI Models (Agent orchestration)
OPENAI_API_KEY=your-openai-key
ANTHROPIC_API_KEY=your-anthropic-key
GOOGLE_API_KEY=your-google-ai-key
```
