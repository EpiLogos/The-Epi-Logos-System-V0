# Story 07.01: archetypal-seeding

## Status
Draft

## Story
**As a** newly signed-up user,
**I want** to provide my birth data and engage in a brief, structured dialogue with the Nara persona,
**so that** my Personal Pratibimba can be automatically seeded with my foundational 'Archetypal Signature' and I can quickly begin my journaling practice.

## Acceptance Criteria

1. Upon first login, the user must be prompted to enter their birth data (date, time, location).
2. The system must use this data to automatically generate the necessary astrological and Human Design information via API calls to our chosen services.
3. Following data entry, the Nara persona must initiate a short, scripted, conversational flow to present key insights from the generated data and co-create the user's initial "Quintessential Identity Structure."
4. This dialogue must feel personal and insightful, not like a robotic survey.
5. Upon completion of the dialogue, the user's Personal Pratibimba must be "seeded" with this foundational signature.
6. The full, multi-layered Identity Matrix section of the app must be available for the user to explore and add to at their own pace later.
7. After the seeding dialogue is complete, the user must be seamlessly transitioned to the main Journal Interface.
